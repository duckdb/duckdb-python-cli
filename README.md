## DuckDB CLI

This package distributes the [DuckDB Command Line Interface](https://duckdb.org/docs/clients/cli/overview) as a Python package.

## Installation

```bash
pip install duckdb-cli
```

## Usage

After installation, the `duckdb` command is available:

```bash
duckdb
```
